# pingfederate_oauth_authorization_detail_processor (Resource)

Resource to create and manage an oauth authorization detail processor.

## Example Usage

```terraform
resource "pingfederate_oauth_authorization_detail_processor" "processor" {
  processor_id = "detailProcessor"
  configuration = {
    fields = [
      {
        name  = "field1",
        value = "value1"
      }
    ]
  }
  name = "Detail Processor"
  plugin_descriptor_ref = {
    id = "com.example.MyAuthorizationDetailProcessor"
  }
}
```
